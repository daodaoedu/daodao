# Mentions Notifications Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 完成 @mention 功能，讓被提及的使用者收到 in-app 和 email 通知。

**Architecture:** Frontend 在使用者選取 @mention 時追蹤 `mentionedUserIds`，submit 時帶入 API。Backend 驗證後對每位被 mention 的使用者建立 `notification_events`（type: `mention`, priority: `P1`），現有的 in-app/email worker 自動處理後續通知。

**Tech Stack:** TypeScript, Zod, Prisma, React, BullMQ (existing workers, no changes)

**Spec:** `docs/superpowers/specs/2026-03-19-mentions-design.md`

- [ ] **Step 4: 在 `comment.service.ts` 加入 mention 通知邏輯**

```ts
// Fire-and-forget: mention 通知給被 @mention 的使用者
const mentionedUserIds = params.mentionedUserIds;
if (mentionedUserIds && mentionedUserIds.length > 0) {
  const ownerId = targetEntity.user_id ?? null;
  const uniqueIds = [...new Set(mentionedUserIds)].filter(
    (id) => id !== userId && id !== ownerId
  );

  if (uniqueIds.length > 0) {
    try {
      const foundUsers = await tx.users.findMany({
        where: { id: { in: uniqueIds } },
        select: { id: true },
      });
      const validIds = foundUsers.map((u) => u.id);

      // targetType → URL path 對照
      const pathSegmentMap: Record<string, string> = {
        check_in: 'check-ins',
        practice:  'practices',
        post:      'posts',
        resource:  'resources',
        note:      'notes',
        outcome:   'outcomes',
        review:    'reviews',
        circle:    'circles',
        idea:      'ideas',
        portfolio: 'portfolios',
      };
      const frontendUrl = process.env.FRONTEND_URL ?? 'https://daodao.cc';
      const segment = pathSegmentMap[targetType];
      const url = segment
        ? `${frontendUrl}/${segment}/${targetId}`
        : frontendUrl; // 未知 targetType → fallback 首頁
      const actorName = createdComment.users.nickname ?? undefined;

      for (const recipientId of validIds) {
        notificationEventService.createEvent({
          type:        'mention',
          actorId:     userId,
          recipientId,
          entityType:  'comment',
          entityId:    createdComment.id,
          priority:    'P1',
          payload:     { content, actorName, url },
        }).catch(() => {});
      }
    } catch {
      // best-effort: mention 失敗不影響留言
    }
  }
}
```
