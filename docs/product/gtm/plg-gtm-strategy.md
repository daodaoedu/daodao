# 島島阿學 GTM 策略 v2：雙軌冷啟動

> 版本：v2（2026-07）——**前提修正版**，取代 v1（v1 見 git history）
> v1 的致命假設：存在一批可轉化的 civic tech 存量活躍用戶。現實：目前幾乎沒有活躍用戶。
> 因此題目從「變現存量」改為「冷啟動」，並依決策調整目標客群：**暫緩實驗教育機構線**，改走雙軌。
> 3 個月目標重定：不是「可預測的 Free→Paid 漏斗」（零用戶做不到），而是**「一個真的在轉的小生態 + 首批付費承諾」**。

---

## Executive Summary

1. **雙軌冷啟動**：軌道 A 直接獲取「有明確動機＋短期明確目標」的個人學習者（日檢、英檢、轉職）；軌道 B 透過社群經營者與自有課程品牌擁有者，把他們既有的付費受眾整批帶進島島。A 給產品驗證與故事，B 給密度與量。
2. **獲客單位是「團」不是「散客」**：養分循環（打卡→被看見→被回應→回流）需要密度，30 個圍繞同一場考試打卡的人才撐得起回應率，300 個散落註冊者撐不起。考試日期是免費的共同 deadline，課程品牌的學員群是現成的密度。
3. **收費分兩端**：學習者端對收成收費（轉職者的資產匯出、AI 額度）；品牌端對「學員成果可見性」收費（陪跑空間、進度儀表板、批次匯出）——課程品牌的痛是完課率與續購，島島賣的是完課引擎。零存量用戶的唯一好處：**沒有「免費既得權」包袱，可以從第一天就定價**，v1 整套社群轉化 playbook 作廢。

## 1. 冷啟動第一性原則

| 原則 | 內容 | 對應 v1 的錯誤 |
|------|------|----------------|
| 密度優先於數量 | 一個 30 人的團勝過 300 個散客；回應率來自「團友情誼」不是陌生人善意 | v1 假設靈感牆自然有回應（因為假設有存量社群） |
| deadline 是免費的引擎 | 日檢/英檢/轉職窗口自帶節律與動機，不需要平台製造 | v1 靠季節內容喚起需求 |
| 先做不規模化的事 | 前 4 週是創辦人手動招募與陪跑，不是行銷排程 | v1 開局就跑內容日曆 |
| 從第一天定價 | 沒有存量就沒有背叛敘事；免費期是「design partner 交換條件」不是「預設狀態」 | v1 花 1/4 篇幅處理轉型情緒 |

**指標戒律不變**：量流動不量壓力——開團機制用考試日期當共同節律，但平台指標不量「達成率」「連續天數」「時長」；考試結果屬於用戶，節律與互相回應屬於島島。

## 2. 目標客群（雙軌，兩個分開的群體）

### 軌道 A｜個人直取：短期明確目標的學習者

**A1 考期學習者（日檢／英檢）**

| 欄位 | 內容 |
|------|------|
| 輪廓 | 20–35 歲，報了 12 月日檢（N3–N1）或近期多益/英檢場次，自學為主、補習為輔 |
| 動機結構 | 考試日期 = 外部 deadline，動機不需要平台維持，需要的是**節律的容器與團友的可見性**（一個人備考的孤獨是真實痛點，Dcard/Threads 滿是「求讀書會」文） |
| 島島的賣點 | 「N2 90 天衝刺團」：複製官方備考實踐模板 → 每日打卡 → 靈感牆看見團友 → 快速回應互相灌能。全部用已上線功能 |
| 付費潛力 | 低（考試本身才是收成，證書不是島島發的）。**角色定位：免費密度來源 + UGC 來源**，變現靠 AI 額度（生成備考行動）與少量升級，不強求 |
| 台灣考期資產 | 日檢一年兩場：12 月第一個週日（報名 8 月底–9 月中）；多益幾乎每月有場次；英檢分季。**考期日曆是內容與開團的節拍器** |

**A2 轉職者**

| 欄位 | 內容 |
|------|------|
| 輪廓 | 27–34 歲，3–9 個月轉職窗口，自學軟體/設計/行銷/日文轉外商 |
| 動機結構 | 自帶目標但缺兩樣：軌跡的組織（散落各工具）與可出示的證明 |
| 島島的賣點 | 打卡軌跡 → 匯出成履歷附件的學習歷程。**這是 A 軌唯一強付費點**，付費時刻 = 面試前 |
| 付費潛力 | 高（NT$149–349/月，轉職焦慮預算），是首批 B2C 付費的主力假設 |
| 與 A1 的交集 | 「多益 800 轉外商」「日檢 N1 轉日商」同時是考期學習者與轉職者——**兩個 persona 的交集是最肥的一塊**，開團主題優先選這裡 |

### 軌道 B｜群體批發：社群經營者與自有課程品牌

| 欄位 | 內容 |
|------|------|
| 輪廓 | 兩類小 B：①學習型社群經營者／讀書會發起人（Threads/IG/FB 上帶讀書會、每期收 NT$300–1,500）②自有課程品牌擁有者（線上課程講師：日語老師、多益名師、轉職導師，客單價 NT$1,500–8,000） |
| 他們的痛 | **完課率與續購**。台灣線上課程完課率普遍低落，課後學員黏著靠講師手工經營 LINE 群，無法規模化；讀書會發起人每期手工追蹤成員進度 |
| 島島的賣點 | 「學員陪跑空間」：講師把課程作業/練習做成島島實踐模板 → 學員複製、每日打卡 → 講師看得見全班節律（不是監控時長，是看見流動）→ 期末批次匯出學員成果頁，成為**下一期招生素材** |
| 為什麼付費 | 對收成收費成立：小 B 的收成 = 學員完課、成果外顯、續購率。島島直接作用在他的營收上 |
| 決策結構 | 個人決策、無採購流程、一通電話能成交——比實驗教育機構快一個數量級，這是取代機構線的核心理由 |
| 帶量能力 | 一個 partner = 一期 15–100 名學員整批進站，**密度即到位**（學員彼此本來就認識） |

**兩軌的關係**：分開獲取、互相餵養。A 軌產出真實個人故事與產品打磨，成為 B 軌的說服素材（「你的學員會得到這種體驗」）；B 軌帶進的學員在期末變成 A 軌的自然用戶。**不是** A 軌用戶由發起人招募——兩軌入口、訊息、成交對象完全分開。

**B 軌 partner 篩選規則（文化守門）**：偏好「學員自己練、自己選題」型的課程與社群（寫作、程式、設計、創作類讀書會），避開純應試解題班——partner 的學員文化會直接長進靈感牆，應試型群體會帶入時長與進度焦慮的度量文化，與指標戒律相斥。

### 2.1 客群評估尺（五維度）與候選池

評估任何新客群用五個維度，**第五維度是否決項**：

1. 動機強度（有沒有自帶的理由持續行動）
2. 自帶密度（是不是一群人、有沒有共同節律）
3. 可賣的收成（有沒有值得付費帶走的東西）——**注意：這是定位維度，不是門檻**。冷啟動期的主目標是活的生態，付費驗證只需要小樣本；收成弱的客群定位為「密度與 UGC 來源」照收，免費用戶的打卡與回應本身就是產出，不是未轉換的遺憾
4. 現有功能貼合度（要不要為它開發新東西）
5. **自主學習 DNA 相容性（唯一否決項）**：學習路徑是不是學習者自己定義的？這個群體的既有文化會強化還是腐蝕島島的指標戒律（量流動不量壓力）？冷啟動期的前 100 個用戶會永久定義島的文化，**帶錯文化的密度比沒有密度更糟**。國考被剔除是因為這一條，不是因為付費力。

| 候選客群 | 判定 | 理由摘要 |
|----------|------|----------|
| 轉職者 | ✅ 現役（A 軌） | 五維全過，自主性最強的付費客群 |
| 日檢/英檢考期學習者 | ✅ 現役（A 軌） | 維度 5 邊緣通過（多為自選目標）；收成弱、定位為免費密度與 UGC 來源 |
| 小 B（課程品牌/社群經營者/讀書會發起人） | ✅ 現役（B 軌） | 依上方篩選規則守門 |
| Build in Public / Product Builder 聚落 | ✅ **Phase 0 並行啟動（創世聚落，見 §2.3）** | 維度 5 全池最純、天然公開打卡文化、自帶擴散；由創辦人「做島島的 90 天」軌跡直接長出，founder-market fit 天然成立。變現預期為零——他們是密度、UGC 與傳教士，不是付費客群 |
| 高中自主學習計畫（108 課綱） | 🗄️ Backlog 首位 | 維度 5 強通過（名字就叫自主學習）、政策製造的剛需、收成同構（軌跡→成果報告）；但決策鏈回到學校體制，冷啟動期不碰 |
| 實驗教育機構/自學家庭 | 🗄️ Backlog | v1 遺產，2027 年 2 月學期窗口前重評 |
| 職涯生態 BD（CakeResume/Yourator） | 🗄️ Phase 2 槓桿 | 機制不是客群；談判週期不可控，不當冷啟動主力 |
| **國考/公職考生** | ❌ **剔除（維度 5 否決）** | 學習路徑完全外部定義（考科、範圍、方法皆標準化），島島核心功能對其多餘；其圈層文化（時長、D-day、進度焦慮）正是指標戒律禁止的度量，冷啟動期湧入會永久改寫靈感牆文化；且無可賣收成（上榜是唯一信號，不是島島發的）。**除非產品哲學改變，否則不重啟** |
| AI 工作應用學習者 | ✅ **已實證拉力（Claude 101 即此群），現役化** | 動機是全池最強的持續順風（職場 AI 焦慮不像考試會過期）；收成中強（「我的 AI 工作流」可展示可寫履歷）；維度 5 通過（工具是教的、應用場景是自己的工作）。**形態為「前共學、後共工」兩段式**：前 2–3 週共讀格式（官方課程教材）→ 後 4–6 週創造格式（週題「本週自動化了什麼」＋期末工作流分享會），是兩種格式的天然橋接。**變現路徑全池最短**：「部門 AI 共學團」即企業讀書會變體的自然落地，C 端跑通兩團後帶案例接企業場；島島本身以 Claude Code 建造，創辦人日常打卡即活教材 |
| 每日打卡社群（LINE 群/百日挑戰） | 🔜 **雙軌補給線** | 唯一「已經在做島島核心行為」的現成群體，功能貼合免教育。真痛點：LINE 群打卡紀錄被聊天洗掉，無法回顧與統計 → A 軌走「軌跡搬家」遷移獲客（鉤子：「你在 LINE 群打的卡，三個月後找得回來嗎？」）；付費挑戰主辦方列 B 軌 partner，免費群主當 design partner——不付費不是扣分，帶密度即價值。**守門：收陪伴型、拒懲罰型**（streak 排行/押金沒收/斷卡踢出文化牴觸指標戒律）；差異化話術：「不算連續天數，斷了也沒關係」 |
| 自我成長（行動導向） | 🔜 **B 軌 partner 主力類型**，A 軌不收 | 維度 5 強通過（目標自定義、與「量流動不量壓力」哲學同頻）、打卡（心情/筆記）功能貼合度全池最高；但動機瀰散無 deadline，散客留存必差 → 不走 A 軌直取，改收「行動導向的成長型讀書會/社群」當 B 軌 partner（台灣說書人/讀書會生態成熟且已在收費）。守門條件：學員要有行動產出可打卡（讀後實作、筆記實踐），純討論/心靈雞湯型社群不收——靈感牆若無行動紀錄，複製鏈無物可複製 |
| 運動賽事備賽/健身自學 | 🗄️ Backlog（結構成立，非現役） | 細分後救回：**賽事備賽**（初馬/半馬 16 週課表 + 固定比賽日）結構上就是考期型衝刺團；**健身自學**（研究訓練學、自排課表）通過「學知識並實作」界線，且是視覺型打卡強題材。剔除線收窄為：無學習內容的純習慣打卡（早起喝水）與體重/體脂排行文化（壓力型度量牴觸戒律）不收 |
| 企業讀書會/自主學習社群 | 🔜 B 軌企業變體（inbound 可收，冷啟動期不主動 outreach） | 與合規型內訓拆開看：福委會/部門有讀書會預算，結構上是 Community 方案換買家，NT$4,900/期對企業是零錢、決策鏈短。**紅線：只收自願參加，拒績效考核掛鉤場景**——儀表板給主揪看節律，不給主管當考核工具，否則「看見流動」淪為監控 |
| 合規型 LMS 內訓（完訓時數/合規報表） | ❌ 剔除 | 無內容、需合規追蹤與時數度量（正面牴觸戒律）、採購流程重，與島島哲學和量能雙重錯位 |

### 2.2 A 軌開團題目候選池（2026-07 已核實考期）

開團模式的公式：**自選目標 × 固定考期（免費 deadline）× 同好文化**。以下依複製成本與貼合度排序（考期均經網路核實於 2026-07-08，開團前仍須向主辦單位二次確認）：

| 順位 | 題目 | 考期/節拍 | 判斷 |
|------|------|-----------|------|
| 現役 1 | 多益 | 幾乎每月有場次，鎖 10 月場 | 第一團（W1–W4 揪團） |
| 現役 2 | 日檢 JLPT | **2026-12-06 考試，報名約 2026 年 8 月**——與計畫 W7 揪團期完全對齊 | 第二團 |
| 3 | 韓檢 TOPIK | **2026-10-18 考試（台北/台中/高雄），報名 6/26–7/13 已截止**——7 月中後存在一批「已報名、備考期 3 個月」的固定母體，**8 月開團揪人剛好收割**，與日檢團並行成本低 | 第三團候選，K-culture 動機純自選、Threads 韓文學習生態活躍 |
| 4 | **留學/升學申請者**（雅思/托福備考只是其中一段） | 真 deadline 是申請季：出國 12–1 月交件、國內研究所推甄備審 9–12 月；7 月開 90 天團，軌跡累積期正好接上文件撰寫期 | **收成全池最強**：打卡軌跡直接變成 SOP/CV/備審資料的自主學習素材，不用解釋用途；付費意願已被代辦（6–20 萬）與補習支出驗證。含兩個子群：出國留學、國內研究所推甄（母數更大、全中文、獲客成本低）。一次性 churn 是結構性的——按季/按收成收費即相容，每屆成功者的公開軌跡是下一屆的揪團素材。**B 軌新類型：留學顧問/代辦**（痛點：學生寫 SOP 時沒有自主學習經歷可寫，島島是顧問指派素材累積的工具）。Dcard 留學板常年「徵讀書會」文，孤獨感需求已被市場自證 |
| 5 | 台語認證（教育部臺灣台語認證、成大全民台語認證） | 教育部場次 3 月/7 月；成大秋季班報名 9/1–9/30 | **DNA 契合度特高的黑馬**：語言復振是文化認同驅動的純自選學習，與島島 civic 出身同源，還有教師採認、縣市獎勵金等外部誘因；母數小但敘事價值大（媒體與 civic 圈天然有故事性） |
| 6 | 無人機操作證（民航局） | 系統隨時排考（rolling），備考 1–2 個月；學科 NT$200/術科 NT$500 | 興趣轉專業、空拍照片是視覺型打卡素材、考後有作品集收成；注意其刷題庫文化偏應試，開團時以「飛行練習紀錄」為主軸而非刷題進度 |
| B 軌轉介 | 手作證照（SCA 咖啡、烘焙丙級、WSET） | 證照與課程綁定（SCA 課程行情 NT$26,600–28,000） | 不適合 A 軌直取（學習發生在教室裡）——**轉為 B 軌 partner 名單**：咖啡教室、烘焙教室是典型「學員自己練、作品視覺化」的小 B |
| 7 | **實踐型長銷書團**（《做自己的生命設計師》《12 週做完一年工作》、神經可塑性/《原子習慣》類） | 無官方考期，但**書自帶時間框架與練習清單**——《12 週做完一年工作》書名就是 deadline，《生命設計師》有奧德賽計畫等具體練習 | 一次解掉兩個懸案：①自我成長客群的 A 軌入口——書就是行動篩選器（實踐型書自帶練習，天然滿足「有行動產出可打卡」的守門條件）②自製 deadline 機制的最佳首發題材（書的方法論與權威取代官方考期）。長銷書 = 長尾流量：書名搜尋量常年穩定，公開實踐頁可吃「書名+心得/練習」SEO，一個好模板永久獲客。**注意**：模板只能寫自己的引導語、不得抄錄書中內容（著作權）；書內建的 scorecard/habit tracker 歸讀者自用，島島平台指標不量它。**B 軌新類型：出版社與說書人**（出版社要書的長尾與社群，「買書入團」是現成的 co-marketing；瓦基類說書人已在辦收費讀書會） |

### 2.2.1 現有營運資產（2026-07 盤點，現實修正）

策略原假設零營運，實際上已有成熟的活動格式在跑（Luma 報名 × Discord 語音 × 島島記錄）：

| 活動 | 形態 | 實證 |
|------|------|------|
| 《Rewire 神經可塑性》共讀（5 月） | 零社交壓力共讀，每週二四晚間，主題實踐記錄 | 實踐型長銷書團（順位 7）**已被驗證過一次** |
| re-Visit 共讀（7 月進行中） | 靜默共讀 50 分鐘＋21:55 打卡儀式 | 「同步共在」繞過空島問題——語音在場感本身就是被看見 |
| Claude 101 共學（7/13 開跑） | 三週完成 Anthropic 免費課程＋證書 | **14 人報名，三活動最強拉力**；AI 共學者是創世聚落的前廳 |

**三層架構**：Luma 是門（報名＋email 名單資產）、Discord 是客廳（同步語音場地）、島島是家（異步軌跡沉澱）。北極星指標從報名數改為**活動→島上持續打卡的轉化率**。三個待補的縫：①複製實踐＋首卡作為開場儀式標準化、21:55 打卡作為固定收尾；②活動結束 48h 的 Luma 名單 email 回流信；③期末軌跡回顧作為收成環節與下期 social proof。

**邊界修正**：「不建 Discord」修正為「Discord 當同步場地、不當異步社群」——語音共在留在 Discord，聊天與記錄留在島上。考期團可直接複用此格式（靜默刷題＋打卡收尾），零新設計。

### 2.3 創世聚落：Product Builder 第一團設計

**論點**：AI 讓產出變便宜後，作品集失效、build log 變成新履歷——軌跡是判斷力與真實性的唯一證明（Mosseri：authenticity is fast becoming a scarce resource）。對學習者島島說「軌跡是學習證明」，對 builder 說「軌跡是判斷力證明」——同一個產品、同一個收成邏輯。

**定位**：不是第二條產品線，是島島的創世聚落——由創辦人自己的軌跡長出的第一批住民。**邊界三條**：變現預期為零；不建島島以外的陣地（無 Discord、無獨立品牌、無線下活動）；不佔行銷預算，唯一新增動作是模板化＋一篇邀請文。

**與現有共讀/共學格式的本質差異（不可直接套用）**：共讀是「一起消費」（教材給定、可排程同步、終點天然、社交需求是共在），builder 團是「一起創造」（無共同教材、工作異步、無天然終點、社交需求是回饋與被看見）。移植三樣：三層架構、打卡儀式、**零壓力哲學**（builder 圈既有社群多為高表演性的 demo/pitch 文化，「零壓力的建造社群」是市場空位）；重造兩樣：①同步環節從共讀改成**共工夜**（每週一次、可選的語音 co-working，異步打卡＋小隊互評才是日常主體——與共讀的同步/異步主從關係相反）②共同軸心從教材改成**週題＋Demo Day**（週題是教科書的替代品，Demo Day 是人工造的終點）。

**Claude 101 是漏斗不是等號**：AI 共學者是學習者身分，課程結束時才是轉化點（「你學會了 Claude API，拿它做一個東西——builder 團 __ 月開跑」），14 人保守估轉化 2–4 人。

**光有模板不夠**——builder 已在 Threads 有受眾，多維護一個地方需要理由。六個機制環環相扣：

| # | 機制 | 設計 | 解決什麼 |
|---|------|------|----------|
| 1 | 團期制＋Demo Day | 不做 rolling 加入：「第一團 90 天，__ 月 __ 日開跑，期末線上 Demo Day」——產品與 90 天軌跡一起在展示廣場發表 | 自製 deadline。無團期＝永遠「改天開始」 |
| 2 | 原型小隊 | 報名時自選 Boris 五原型（Prototyper/Builder/Sweeper/Grower/Maintainer），依原型混編 4–5 人小隊互關 | 回應的結構性保證＋回應的品質——跨原型視角（Prototyper 收到 Sweeper 的回饋）比隨機打氣有價值 |
| 3 | 週題（weekly prompt） | 每週一個原型化題目：「本週的 Sweeper 時刻——你砍掉了什麼？」「Grower 時刻——哪個數字動了？」 | 降低「今天要寫什麼」的門檻，讓全團內容可互相對照、天然成為 Threads 轉發格式 |
| 4 | 創辦人對等入團 | 島島本身就是團內被 build 的產品之一，創辦人以成員身分打卡，不以主辦者身分管理 | Skin in the game——對等的問責比主辦者的經營更有黏性；創辦人先打滿 14 天卡才發邀請（模板頁上看得到活的示範） |
| 5 | Threads 相容定位 | 「Threads 是你的舞台，島島是你的後台」——貼文會被滑掉，log 會累積；打卡內容可自行轉發 Threads（W8 後有 OG 分享卡） | 不與既有受眾陣地競爭，做沉澱層而非另一個發布平台 |
| 6 | 期末收成 | 90 天軌跡回顧頁——builder 圈的高流量內容格式就是「產品發布＋開發背後的 90 天」，回顧頁就是那篇文的素材 | 收成時刻與 launch 時刻重合（對收成收費原則的免費版示範） |

**成功判準**：第一團 ≥10 個 builder 完成 ≥30 天紀錄、Demo Day ≥5 個產品發表。**失敗判準**：邀請文發出兩週 <5 人複製 → 不追加行銷，聚落退回「創辦人個人軌跡」形態，等多益團的密度養起來再試第二次。

**順帶收穫（B 軌 outreach 名單種子）**：搜尋過程確認的台灣線上課程品牌生態——日語（王可樂日語、TKB 等，N2 課程行情 NT$16,000–17,000）、語言家教平台（AmazingTalker 上的獨立老師）、咖啡/烘焙教室——皆為 B 軌 partner 的候選類型，完課率痛點與客單價足以支撐 Community 方案定價。

**暫緩清單**：實驗教育機構與自學家庭（v1 P2/P4）移入 backlog——不是否定，是冷啟動期資源只夠雙軌；審議會報告書季（11–12 月）若屆時有餘裕可低成本試水。企業內訓維持不碰。

## 4. 方案與定價（v2 調整）

四級框架保留，內容重排。鐵律不變：對收成收費、不對裝飾收費；養分循環（打卡、回應、靈感牆、複製、治理）全免費。

## 5. 指標框架（v2：以團為單位）

三層不變（生態→收成→商業），分析單位從全站改為「團」——全站平均在 150 人規模下沒有統計意義，團與團的對比才有。

| 層 | 指標 | 定義 | 12 週目標 |
|----|------|------|-----------|
| 生態 | 團內回應率 | 團內公開打卡 72h 內獲 ≥1 回應的比例 | 建基線，健康線假設 ≥40% |
| 生態 | 團內回流率 | 收到回應者 7 日內再打卡比例 | 基線 |
| 生態 | 團週活躍率 | 當週有 ≥1 次打卡的團員比例（**不是連續天數**） | ≥50%（B 端 partner 談續約的核心數據） |
| 生態 | 模板複製數 | 官方/講師模板被複製次數（=入團數） | 3 團合計 ≥100 |
| 收成 | 匯出次數 | 免費+付費匯出 | ≥50 |
| 收成 | 考後分享 | 軌跡回顧頁分享數 | ≥20 |
| 商業 | B 端付費續期 | partner 續期且付費 | ≥1 |
| 商業 | B2C 付費數 | Basic/Premium 訂閱 | ≥10 |
| 哨兵 | 講師依賴度 | 團內回應中來自講師/官方的佔比 | **應隨週次下降**——若第 8 週還 >50%，團友互回應是假的，生態沒長出來 |
