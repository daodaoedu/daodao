# 寫 SOP／備審時沒有自主學習經歷可寫

TA 原話語言：「學生寫 SOP 時沒有自主學習經歷可寫」；「備審／SOP 的自主學習段落怎麼寫」。

留學與研究所推甄申請者在寫申請文件時，缺乏可具體佐證的自主學習過程——審查委員想看的是紀錄不是形容詞。島島的打卡軌跡直接變成 SOP／CV／備審的自主學習素材（收成全池最強，[[valueprop-learning-portfolio]]）。B 軌衍生新類型：留學顧問／代辦（島島是顧問指派學生累積素材的工具）。此痛暫掛 [[persona-a2]]，但申請者其實是尚未獨立建模的 persona（見存疑點）。
