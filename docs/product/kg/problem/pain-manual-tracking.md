# 手工追蹤學員進度／LINE 群手工經營

TA 原話語言：
- 「課後學員黏著靠講師手工經營 LINE 群，無法規模化。」
- 「讀書會發起人每期手工追蹤成員進度。」

[[persona-b1]]、[[persona-b2]] 為了維繫課後黏著，只能靠人力在 LINE 群一個個追、每期重來，無法規模化。外部證言補強：講師被迫在學員習慣分散的多個現成工具（LINE／Discord／IG）手工經營（[[signal-teachify-completion-below-5]]）；社群主揪長期「硬撐群組熱度」耗竭到退群、且「你退了整個群的氣場就散了」（[[signal-fb-group-host-burnout]]）——證實經營疲勞真實且不可持續。註：目前直接證言集中在「LINE／社群手工經營疲勞」；「逐一手工追蹤每位成員進度」仍偏推論。島島的解方是講師節律儀表板（看流動不看時長，[[feature-b-dashboard]]）——第 3 週就從節律面板看見誰停了、私訊關心，而不是期末才發現（[[valueprop-completion-engine]]）。
