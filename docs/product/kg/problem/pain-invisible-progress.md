# 看不見自己的累積：學了很久卻感覺不到進步、回頭看不到軌跡

TA 原話語言：「看了很多書、上了很多課程，但始終覺得自己沒有進步」；「上完課就真的只是『上完』而已，沒有紀錄、沒有複習、沒有應用」；「筆記只會存進去，從不把你考出來」。

自學者（考期學習者與轉職者皆有）在學習當下感覺不到進步，回頭也看不到自己走過的軌跡——努力是隱形的，容易在「失望之谷」放棄。這不是缺乏學習量，而是缺乏一個把累積「浮現出來」的機制。街上的 DIY 解法是 streak／habit app（[[competitor-habit-app-streak]]）用連續天數硬造可見感，但一斷就歸零、也看不出質的成長。

島島的解方直接對上這個縫：學習週報（[[feature-weekly-report]]）每週把「隱形努力」轉成「可見進度」＋社交反饋；學習歷程匯出（[[feature-export]]）把散落軌跡一次攤開、看見走了多遠。此痛跨 [[persona-a1]]／[[persona-a2]]，是週報功能過去在 KG 內沒有任何痛點支撐（過度設計候選）後補上的需求立論。標 evidenced：三則含 TA 原話的外部 Signal（部落格／Threads／FB），非文件斷言。
