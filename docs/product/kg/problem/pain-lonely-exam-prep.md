# 一個人備考的孤獨

TA 原話語言：
- 「準備考試最孤獨的時刻，是對完答案的那個晚上。分數不上不下，沒有人可以說。」
- 「一個人備考的孤獨是真實痛點，Dcard／Threads 滿是『求讀書會』文。」
- 「你不是沒毅力，你是沒觀眾。」

考試日期已提供 deadline 與動機，缺的是**節律的容器與團友的可見性**。現有替代方案是自組 LINE 讀書會群、Discord 自習室。產品側健檢（[[signal-notification-infra-healthcheck]]）證實通知鏈路已備好、缺的是密度而非功能——解方在團內密度＋衝刺團的團友可見性（[[valueprop-sprint-group-density]]）與人肉保底回應。

外部證據：Dcard 日語板考前徵讀書夥伴、自陳「怕孤軍奮戰、我好不自律」（[[signal-dcard-n2-study-buddy]]）補上原本只有策略宣稱、無 URL 的 [[signal-studygroup-request-posts]]；台灣閱讀 KOL 瓦基把「缺乏陪伴感、孤軍奮戰」列為習慣追蹤失敗五大原因之一（[[signal-readingoutpost-companionship]]），佐證陪伴缺席在自學情境的普遍性。
