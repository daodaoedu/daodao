# 一個人讀不完、讀完沒有累積感

自我成長型讀者（[[persona-a3]]）的雙面痛：一是「一個人讀不完、還把讀不完歸咎自己」，二是「就算讀完也留不下什麼、沒有累積感」。非應試情境。

TA 原話語言：
- 「為什麼有些人一年可以讀完幾十本書，我們卻連一本都讀不完？」「我們太容易把『讀不完』這件事，怪到自己身上。」（[[signal-readingoutpost-cant-finish-book]]）
- 「書架上滿滿的書……合上書一個禮拜後，問他這本書到底改變了什麼，他往往答不上來。」「想讀書但讀不下去、讀完就忘。」（[[signal-vista-read-but-forget]]）
- 「看過很多書……但『知道』卻做不到」「單看書真的很難實際運用」；市場已用付費「不讀書的讀書會」填補（[[signal-accupass-not-reading-bookclub]]）。

三筆獨立證據皆指向同一結構：一個人的閱讀難以持續、且缺乏可見的累積。今天由陪伴型／引導型讀書會（付費給主揪）承接，但仍靠主揪人力。島島的機制假設——實踐追蹤（[[feature-checkin]]）＋衝刺團陪伴（[[feature-sprint-group]]）＋學習歷程可回顧與匯出（[[feature-export]]）——正對應此痛：把「讀了沒留下」變成「被看見、可累積」。solved_today_by 暫留空：現成替代方案（一般讀書會）尚無對應 Competitor 節點。
