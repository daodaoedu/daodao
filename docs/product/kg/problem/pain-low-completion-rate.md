# 學員完課率低

TA 原話語言：
- 「課賣出去之後，你看不見誰真的在練。學員上完課回到自己的生活，練習斷了、群組安靜了，下一期續購時你只能重新說服一次。」
- 「完課率不只是教學品質指標——它直接決定續購率與口碑。」
- 「完課率低不是學員懶，是課後那八週沒有人陪。」

台灣線上課程完課率普遍低落，對 [[persona-b1]]、[[persona-b2]] 直接衝擊營收（續購率）。P2 外部研究已把先前僅靠內部斷言（[[signal-low-completion-rate-taiwan]]，以 `[DATA]` 佔位）的完課率補成三方交叉印證的量級：自學型線上課完課率多數低於 5%（[[signal-teachify-completion-below-5]]）、台灣完課率約 10–20%／人均 1.27 堂（[[signal-threads-completion-10percent]]）、「買了課有一半人連第一章都沒打開」（[[signal-fb-completion-first-chapter]]）。且證言明確指出真正代價是「沒有學員見證、口碑與轉介紹→下一檔賣不動」，而非退款。島島賣的是完課引擎（[[valueprop-completion-engine]]）：課後練習做成實踐模板、學員複製打卡、彼此看見、講師看得見全班節律。
