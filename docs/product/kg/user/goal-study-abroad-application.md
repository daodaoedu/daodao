# 留學／研究所申請

完成留學或國內研究所推甄的申請文件（SOP／CV／備審）。含兩個子群：出國留學（12–1 月交件）、國內研究所推甄（9–12 月書審）。

真 deadline 是**申請季**而非某場考試（雅思／托福備考只是其中一段），故 `anchored_by` 留空。**收成全池最強**：打卡軌跡直接變成 SOP／備審的自主學習素材，對應痛點 [[pain-sop-no-selflearning-experience]]。付費意願已被代辦（6–20 萬）驗證。此目標暫掛在 [[persona-a2]] 下，但申請者其實是一個尚未獨立建模的 persona（見存疑點）。
