# A3 自我成長型讀者

參加陪伴型／創作選書型讀書會的成人讀者，目標不是考試或轉職，而是持續閱讀與自我成長本身。

動機結構：想持續讀但一個人難以持續，付費參加讀書會買的是**陪伴、節律與被主揪看見**（每期 NT$300–1,500 付給主揪，非直接付島島）。零應試味，與島島「不量壓力、量流動」的指標戒律天然同調。

GTM 角色：經由 B1 社群經營者（如 [[org-miartist-studio]]、[[org-min-yu916]]）整批帶入的密度來源。BD 名單中文化最契合的兩個讀書會候選受眾都是這型，先前因無 canonical persona 被錯掛 a2。

痛點驗證（P2 完成）：痛點輪廓已由三筆獨立外部證據支持，新建 [[pain-read-alone-no-accumulation]]（一個人讀不完、讀完沒有累積感）。證據：閱讀前哨站瓦基「別人一年讀幾十本、我們連一本都讀不完，還怪自己」（[[signal-readingoutpost-cant-finish-book]]）、資深讀書會主持人 Vista「書架滿滿、讀完就忘、沒留下什麼」（[[signal-vista-read-but-forget]]）、付費「不讀書的讀書會」證實此受眾買的是陪伴而非知識且願付費（[[signal-accupass-not-reading-bookclub]]）。故 confidence 升為 evidenced。

研究欠債（剩餘）：仍只有 1 個 evidenced PainPoint（schema 建議 ≥3）。belongs_to／pursues／pays_at 尚無對應節點可掛（現無「陪伴型閱讀」segment 與付費 moment 節點），待後續補建。
