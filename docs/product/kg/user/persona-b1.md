# B1 學習型社群經營者／讀書會發起人

在 Threads／IG／FB 上帶讀書會、每期收 NT$300–1,500 的社群經營者與讀書會發起人。

痛點：每期手工追蹤成員進度（[[pain-manual-tracking]]）、成員讀完就散、完課率低（[[pain-low-completion-rate]]）。島島的賣點是「學員陪跑空間」與講師節律儀表板（[[valueprop-completion-engine]]），期末批次匯出成果頁成為下一期招生素材。個人決策、一通電話能成交，付費時刻是續期（[[payment-b-community-renewal]]）。partner 篩選偏好「學員自己練、自己選題」型社群，避開純應試班。
