# 「維持群組熱度，付出太多本該用來成長的心力」——群組主揪耗竭

貼文〈她離開那些群組的那一天〉（作者秦愛得，發於 FB 社團「好書共讀」，2025-07-09，685 reactions／272 shares），描寫一位長期扮演「讓群組活過來」角色的社群經營者最後退群。

TA 原話語言（照抄）：
- 「她不是那種默默潛水的人，反而總是那個讓群組活過來的人……總會主動帶點話題、講幹話、貼梗圖、拉人接話、化解冷場的人。」
- 「群組裡的風景總是重複：有人只回『哈哈』，有人永遠已讀，有人發了訊息就像石沉大海。沒有真正的對話，沒有真正的關心。」
- 「為了維持群組的熱度，她付出了太多本該拿來沉澱、進步、成長的時間與心力。」
- 熱門留言：「心累是真的，因為太清醒、太會照顧場面的人，常常自己沒人照顧。你退了，整個群的氣場都散了。」

佐證 [[pain-manual-tracking]] 的「課後學員黏著靠人手工經營 LINE／社群，無法規模化」一面：讀書會／學習社群發起人（[[persona-b1]]）靠一己之力硬撐群組活躍度，情緒與時間成本高、且不可持續（最終退群）。註：此為「經營疲勞」的第一手證言；「逐一手工追蹤成員進度」目前仍屬推論、未取得同等直接原話。
