# 語言檢定備考

自學為主、補習為輔，為某一場語言檢定（日檢／多益／英檢／韓檢）在固定考期前衝刺的學習者所構成的市場。

考試日期是**免費的共同 deadline**，自帶節律與動機——這是 GTM v2 的節拍器。此區隔的核心痛是 [[pain-lonely-exam-prep]]（一個人備考的孤獨），現有替代方案多為自組讀書會與自習室。主要對應 [[persona-a1]]。
