# LINE 群太多、重要訊息「看過就忘、難集中保存」的民間解法

TA 原話語言（FB 貼文，讀者提問與回覆）：
- 阿金：「Line朋友、群組太多了，常常看過了就忘了，要怎麼集中保存重要的訊息？」
- 太座：「你可以設『自己一個人』的群組啊！把重要的訊息，都轉給『自己』的群組。」

佐證 [[pain-record-lost-in-line-group]] 的底層機制：LINE 群訊息量一大，重要內容（含打卡）看過即淹沒、無處集中沉澱，使用者只能用「轉給自己的群組」這種土法煉鋼保存——正說明缺一個把散落紀錄集中回顧的容器。observed_date 無法從貼文確認精確日期，記為抓取日 2026-07-12。
