# 「書架滿滿、合上一週卻答不出改變了什麼」——讀完沒有累積感

資深作家、長年讀書會主持人 Vista Cheng（鄭緯筌）部落格文章（2026-04-29）。作者自述「帶過從十人小聚到上百人線上同步共讀」，兼具讀者痛點與 [[persona-b1]] 主揪視角。

TA 原話語言（照抄）：
- 「我看過太多人讀很多但派不上用場的場景：書架上滿滿的書，用螢光筆畫了一堆金句，看起來很勤奮，但合上書一個禮拜後，問他這本書到底改變了什麼，他往往答不上來。」
- 「特別推薦給『想讀書但讀不下去』『讀完就忘』的朋友。」
- 「如果你跟我一樣，書架上書一堆、心裡卻常常覺得讀了好像也沒留下什麼……」
- 主揪視角規律：「在讀書會中願意分享、願意輸出的成員，三個月後對書的記憶與運用程度，遠遠高過只是默默聽的成員。輸出不是讀書的副產品，而是讀書本身。」

佐證 [[pain-read-alone-no-accumulation]]：自我成長讀者（[[persona-a3]]）的另一面痛——「讀完就忘、沒有累積感」，且作者以主揪身分點出「有輸出／被看見的成員留存與運用度更高」，直接對應島島「打卡＋彼此看見＋歷程可回顧」的機制假設。
