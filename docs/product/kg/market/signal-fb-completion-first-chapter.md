# 「買了課，一半的人連第一章都沒打開」（引述某線上課平台老闆）

FB 粉專「雪倫的隱藏版生活」貼文（2026-04-18），引述其線上課平台老闆友人的內部數據。

TA 原話語言（照抄）：
- 「台灣線上課的完課率，大概只有 20%。甚至買了課之後，有一半的人連第一章都沒打開。」
- 平台老闆原話：「台灣的線上課，完課率大概只有 20%。甚至買了課之後，真的打開第一章的人，可能都不到 50%。……一半的人，買完就沒有開始。八成的人，開始了也沒有走完。」
- 「有時候他覺得自己不是在賣課，是在賣一個『安慰劑』。」
- 「如果你上完一堂課，沒有去實踐，大概一個禮拜之後，80% 的內容你都會忘記。真正的學習，是你開始做之後才開始的。」

佐證 [[pain-low-completion-rate]]：正是策略文件想要的「學員上完第一章就消失」原話，且來自 [[persona-b2]] 圈內（課程平台老闆）的實務數據，與 [[signal-teachify-completion-below-5]]、[[signal-threads-completion-10percent]] 三方交叉印證台灣完課率量級。
