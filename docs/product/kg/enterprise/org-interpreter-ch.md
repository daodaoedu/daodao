# 中韓口筆譯員（@interpreter_ch）

自有課程品牌講師中**文化契合最佳**者，經營陪伴型、養成型的中韓視譯小班，非解題班。

- 受眾規模：Threads 1,441 粉，粉絲數小但**學員黏著度極高**（學員會主動推薦她）。
- 為什麼是好候選：貼文提到「視譯小班**第一期**就一路到現在的學員」「不只要把你們送進梨大」——**已在辦第 2 期以上**，命中 `w1-w2-execution-pack.md:78` 描述的對象。小班制最痛的是學員之間看不到彼此的練習與講師手工追蹤進度，對應 [[pain-manual-tracking]]；養成型長週期也吃 [[pain-low-completion-rate]]。
- 受眾對應 persona：學員為韓文／口筆譯學習者，對應 [[persona-a1]]（部分朝梨大升學方向，帶 [[persona-a2]] 色彩，但主體是語言學習）。
- 開場角度：小班制最痛的是「學員之間看不到彼此的練習」。島島讓同團學員的 log 互相看得見。
