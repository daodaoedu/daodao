# 衝刺團密度與團友可見性

對 [[persona-a1]] 的賣點：把一個人備考的孤獨，改造成一群人圍著同一場考試打卡的密度。

「N2 90 天衝刺團」：複製官方備考實踐模板 → 每日打卡 → 靈感牆看見團友 → 快速回應互相灌能。人少＝每則紀錄都被接住（「你今天的打卡，明天一定有人回」）。回應痛點 [[pain-lonely-exam-prep]]、[[pain-streak-punishes-life]]。`monetizes` 留空——A1 付費力低，此賣點的角色是免費密度與 UGC 來源，變現另走 [[valueprop-ai-study-actions]]。
