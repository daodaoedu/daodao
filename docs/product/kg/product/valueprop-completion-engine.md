# 完課引擎

對 [[persona-b1]]、[[persona-b2]] 的賣點：島島是一台直接作用在**完課率與續購**上的引擎。

把課後練習／共讀進度做成實踐模板 → 學員一鍵複製、每天打卡 → 學員彼此看見互相回應（講師不再是唯一動力來源）→ 講師從節律儀表板看見全班流動（不是監控時數）→ 期末批次匯出全班成果頁，成為下一期招生素材。對收成收費成立（小 B 的收成＝學員完課、成果外顯、續購率）。回應痛點 [[pain-low-completion-rate]]、[[pain-manual-tracking]]，變現於 [[payment-b-community-renewal]]。
